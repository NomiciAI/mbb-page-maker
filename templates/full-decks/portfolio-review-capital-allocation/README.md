# Portfolio Review Capital Allocation

## Summary

Portfolio review and capital allocation archetype for comparing businesses, products, regions, or initiatives and deciding where to build, hold, harvest, or exit.

## Visual traits

Mono capital allocation style, portfolio health metrics, growth/profitability matrix, option scorecard, capital sensitivity, and allocation decision log.

## Use when

Use for business portfolio reviews, capital allocation, product portfolio strategy, resource rebalancing, build/hold/exit choices, and annual strategy refreshes.

## Do not use when

Do not use for single-initiative recommendations, market entry, or investment committee memos unless the main question is portfolio allocation.

## Storyline pattern

Cover, portfolio answer, performance snapshot, portfolio map, strategic roles, capital choices, sensitivity and risks, allocation decisions and next review.

## Layout + component stack

Uses `metric-strip`, `portfolio-prioritization-matrix`, `comparison-table`, `sensitivity-grid`, `risk-matrix-register`, and `decision-log`.

## Theme / assets

Theme: `assets/themes/mono.css`. No required media assets.

## Source inspiration

Source-neutral portfolio strategy and capital allocation patterns. No external deck, brand system, or consulting-firm identity is copied.

## Evidence shape

Best with portfolio units, growth, margin, capital intensity, strategic fit, risk, scenario assumptions, and allocation choices.

## Flexible swaps

Swap portfolio matrix for scatter plot when both axes are numeric, replace sensitivity grid with comparison table when scenarios are unavailable, and replace decision log with roadmap when sequencing is the main ask.

## When to diverge

Diverge when the user needs a single business-case approval, market-entry strategy, or M&A decision rather than portfolio allocation.

## Do not copy page order/content

Do not copy the archetype page order, placeholder portfolio units, footer text, or allocation labels. Use it only to understand portfolio-review composition.
