# Initiative Prioritization Recommendation

## Summary

Initiative recommendation archetype for ranking a pool of initiatives, choosing what to fund, sequencing delivery, and making resource tradeoffs explicit.

## Visual traits

Blue decision style, portfolio matrix, weighted scorecard, dependency and business-case pages, roadmap, and decision ask.

## Use when

Use for initiative prioritization, resource allocation, transformation backlog choices, value-realization planning, and strategic initiative recommendations.

## Do not use when

Do not use for pure project status updates, detailed PMO task plans, or vendor selection unless initiatives are the decision unit.

## Storyline pattern

Cover, recommendation answer, initiative inventory, prioritization lens, business case, dependencies and risks, roadmap, decisions and funding ask.

## Layout + component stack

Uses `portfolio-prioritization-matrix`, `weighted-scorecard`, `sensitivity-grid`, `risk-matrix-register`, `phase-roadmap`, and `decision-log`.

## Theme / assets

Theme: `assets/themes/blue.css`. No required media assets.

## Source inspiration

Source-neutral strategic prioritization and initiative portfolio patterns. No external deck, brand system, or consulting-firm identity is copied.

## Evidence shape

Best with initiative names, value estimates, feasibility/readiness, criteria scores, capacity or funding constraints, dependencies, risks, and decision asks.

## Path

`templates/full-decks/initiative-prioritization-recommendation/index.html`

## Related showcase patterns

`templates/showcase/initiative-portfolio-prioritization-page.html`, `templates/showcase/business-case-sensitivity-page.html`, `templates/showcase/roadmap-risk-decision-page.html`

## Flexible swaps

Swap portfolio matrix for weighted scorecard when criteria are explicit, replace sensitivity grid with comparison table when benefits are qualitative, and replace roadmap with milestone track when status is the main question.

## When to diverge

Diverge when the user needs a full operating plan, investment committee memo, or transformation narrative rather than initiative choice and sequencing.

## Do not copy page order/content

Do not copy the archetype page order, placeholder initiatives, footer text, or scoring labels. Use it only to understand prioritization-story composition.
